# Find My Bike
This application lists all the bikes that have been registered as stolen in a given location provided by the user in the past week. Or it could display statistics 

#### By Morgan Waites, Ella Tanttu & Karen Axon

## Technologies Used

* HTML
* CSS
* Bootstrap
* Node
* Jest
* Webpack
* Babel
* Bike Index API

## Setup/Installation Requirements

* Click on the [repository's](https://github.com/karenaxon/bikes.git) link.
* Click on the green "Code" button and copy the repository URL.
* Open your terminal and navigate to the location where you would like to clone the application.
* Use the command _git clone https://github.com/karenaxon/bikes.git_ to clone the repository.
* From the top level directory, use the command _npm install_ to download the npm packages that the application needs.
* Build the application with the command _npm run build_.		
* Start the application in a server with the command _npm run start_.		
* Open the application in your favorite text editor to make changes.
